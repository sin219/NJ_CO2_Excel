# NJ_CO2_Excel

# Environmental & Population Impact Analysis (Excel)

This project analyzes CO₂ emissions across U.S. states, sectors, and years using Excel-based quantitative analysis.

## Overview
- Evaluates CO₂ emissions by **sector, state, and time**
- Applies **ratio-based metrics** to standardize comparisons
- Uses slicers to support interactive exploration of trends

## Tools & Methods
- Microsoft Excel
- PivotTables and PivotCharts
- Calculated fields and aggregations
- Slicer-driven analysis
